# Налаштування середовища розробки (Jakarta EE 11 Lab)

Цей документ містить інструкції з налаштування єдиного стилю коду (Google Style) та локального сервера Tomcat для коректного запуску проєкту.

## Структура ресурсів
Всі необхідні інструменти знаходяться в папці `tools/` у корені проєкту:
- **Tomcat:** `tools/apache-tomcat-11.0.18`
- **Code Style:** `tools/intellij-java-google-style.xml`

---

## 1. Налаштування Java Code Style

Для уникнення конфліктів при злитті гілок (merge conflicts) всі учасники команди повинні використовувати єдиний стиль форматування коду.

### Для IntelliJ IDEA

1. Відкрийте налаштування: **File** -> **Settings** (або `Ctrl+Alt+S`).
2. Перейдіть у розділ: **Editor** -> **Code Style** -> **Java**.
3. Натисніть на іконку шестерні ⚙️ поруч із випадаючим списком **Scheme**.
4. Оберіть: **Import Scheme** -> **IntelliJ IDEA code style XML**.
5. У провіднику вкажіть шлях до файлу в проєкті:  
   `tools/intellij-java-google-style.xml`
6. Натисніть **OK**, потім **Apply**.

> Тепер ви можете форматувати будь-який файл комбінацією `Ctrl+Alt+L`.

---

## 2. Налаштування запуску (Run Configuration)

Щоб сайт відкривався за адресою `http://localhost:8080/` (без зайвих префіксів у URL), необхідно налаштувати контекст деплою.

### Інструкція для IntelliJ IDEA:

1. На верхній панелі натисніть на випадаючий список конфігурацій і оберіть **Edit Configurations...**.
2. Якщо конфігурація ще не створена:
    - Натисніть `+` -> **Tomcat Server** -> **Local**.
    - У полі **Application server** натисніть `Configure...` і вкажіть шлях до розпакованого томката: `[шлях_до_проєкту]/tools/apache-tomcat-11.0.18`.
3. Перейдіть на вкладку **Deployment**.
    - Переконайтеся, що додано артефакт `jakarta-labs:war exploded`.
    - Внизу знайдіть поле **Application context**.
    - Змініть значення на: `/`.
4. Перейдіть на вкладку **Server**.
    - У полі **URL** має бути: `http://localhost:8080/`.
    - У розділі **On 'Update' action** оберіть `Redeploy`.
5. Натисніть **Apply** та **OK**.

### Перевірка
Запустіть проєкт (Shift+F10). Браузер має автоматично відкрити сторінку за адресою `http://localhost:8080/`.
